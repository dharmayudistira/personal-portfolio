# Design System Document

This document outlines the core design principles and token values for our user interface.

## Branding & Identity

### Color Palette
Our color palette is defined by a striking, monochromatic dark theme.
- **Primary Color**: `#FFFFFF` (White) - Used for prominent actions, text, and active states to provide strong contrast.
- **Secondary Color**: `#2A2A2A` (Dark Gray) - Supports the primary color for less prominent UI elements, subtle backgrounds, and secondary actions.
- **Tertiary Color**: `#FFFFFF` (White) - Acts as an additional accent, consistent with the primary, for highlights or decorative elements.
- **Neutral Color**: `#131313` (Very Dark Gray) - The foundational background and surface color, setting the overall dark tone.

### Typography
Our typography is designed for clarity and modern appeal.
- **Headlines**: `Space Grotesk` - A modern, geometric sans-serif, ideal for strong headlines.
- **Body Text**: `Inter` - A highly legible and versatile sans-serif, suitable for long-form content.
- **Labels**: `Inter` - Consistent with body text for clear and concise labeling.

## User Interface Elements

### Roundedness
UI elements feature a subtle roundedness, with corners having a radius of `1`. This provides a modern yet not overly soft aesthetic, balancing sharpness with a touch of approachable softness.

### Spacing
The design employs a normal to spacious level of whitespace, set at `2`. This ensures good readability and a clean, uncluttered layout, enhancing the user's perception of content.
